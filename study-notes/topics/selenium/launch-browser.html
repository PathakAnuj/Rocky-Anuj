Launching Different Browsers in Selenium

In Selenium WebDriver, you can launch multiple types of browsers using their respective driver classes. Below are the commonly used browsers:

1. Chrome Browser

Use ChromeDriver to launch Google Chrome.

WebDriver driver = new ChromeDriver();
driver.get("https://www.google.com");
driver.manage().window().maximize();
driver.quit();

2. Firefox Browser

Use FirefoxDriver to launch Mozilla Firefox.

WebDriver driver = new FirefoxDriver();
driver.get("https://www.mozilla.org");
driver.manage().window().maximize();
driver.quit();

3. Microsoft Edge

Use EdgeDriver to launch Microsoft Edge.

WebDriver driver = new EdgeDriver();
driver.get("https://www.microsoft.com");
driver.manage().window().maximize();
driver.quit();

4. Internet Explorer

Use InternetExplorerDriver to launch Internet Explorer.

WebDriver driver = new InternetExplorerDriver();
driver.get("https://www.microsoft.com");
driver.manage().window().maximize();
driver.quit();

5. Browser Options

You can also customize browser settings using Options class.

Chrome Example with Options:

ChromeOptions options = new ChromeOptions();
options.addArguments("--incognito");
options.addArguments("--start-maximized");
WebDriver driver = new ChromeDriver(options);
driver.get("https://www.google.com");
driver.quit();

Firefox Example with Options:

FirefoxOptions options = new FirefoxOptions();
options.setHeadless(true); // run without UI
WebDriver driver = new FirefoxDriver(options);
driver.get("https://www.mozilla.org");
driver.quit();

Notes:

  • Ensure the respective driver executable (chromedriver, geckodriver, msedgedriver, IEDriverServer) is available in your system PATH.
  • Use driver.manage().window().maximize() to open browser in full screen.
  • Headless mode allows running browsers without opening a UI window (useful for CI/CD).
